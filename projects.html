Coursework & Group Projects

Here are some recent projects I have done for coursework!

Academic Tracking App (CSE 110 Software Engineering)

A group project worked on throughout the quarter to track class progress: The app lets students:

  1. List the classes they are enrolled in
  2. Add assignments, projects, and quizzes for each class
  3. Record due dates, hours worked, and percent progress
  4. All while also accounting for late date / late policies specific to the classes

Primary skills used:

JavaScript Agile teamwork

Pilot Pichu (CSE 165 3D User Interaction)

A VR game where the user is a pilot whose goal is to finish the race course they are spawned in front of.

What was implemented:

  1. Hand tracking to indicate the go, stop, and turn functions of the pilot in game
  2. World waypoints to act as checkpoints where the player has to go
  3. A way for checkpoints to turn off once reached
  4. Visible timer to record and show player their time
  5. Respawning character if they crashed at last checkpoint hit

Primary skills used:

C++ VR locomotion

20 Questions vs. an AI Bot (CSE 165 #d User Interaction)

A game of 20 Questions played against an AI bot. The bot thinks of a random noun, and the player can either speak or type a yes-or-no question in chat. The bot answers accordingly and will even give hints when asked.

What was implemented:

  1. Voice input- taking in user's voice turning into json for AI to discern
  2. Gestures by AI bot as reactions to questions to create more immersive experience
  3. Ties to virtual keyboard to json so AI can answer given questions

Primary skills used:

Voice input AI integration C++
